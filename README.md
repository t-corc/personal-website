# Tommaso Corciulo Portfolio

A dependency-free personal portfolio focused on the intersection of sales,
automation, and AI-assisted software development.

## Run locally

```bash
npm run dev
```

Then open `http://127.0.0.1:4173`.

## Structure

- `index.html`: content and page structure
- `styles.css`: visual design and responsive layouts
- `script.js`: interactive conversation
- `assets/`: CV and future project assets

The site is static and can be deployed to GitHub Pages, Netlify, Vercel, or any
standard web host.
